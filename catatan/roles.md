# catatan penting
## catatan Saat connect MetaMask,atau wallet kamu hanya melakukan:
- izin aplikasi membaca alamat wallet
- izin aplikasi membaca chain
- tidak ada signature
- tidak ada otentikasi
- tidak ada verifikasi user

## catatan saat connect web3 dibutuhkan :
➡️ authenticated (logged in)
➡️ Web3 login / SIWE / signature login

untuk mendapatkan bukti bahwa user benar-benar pemilik wallet
- bisa membuat session login
- bisa autentikasi user ke backend
- bisa membuat user account di database


===========================================================================
catatan: perbedaan ketika login wallet tidak memerlukan sign dan tidak ada biaya gas, ketika login web3 memerlukan sign sebagai verifikasi pengganti password dan melakukan transaksi dan lainya pada web 3
